# MiniCRM Mobile (PWA) — v1.1.0

Дата сборки: 2025-11-01

## Состав
- `index.html` — приложение (HTML/CSS/JS в одном файле) + регистрация Service Worker
- `manifest.webmanifest` — манифест PWA
- `service-worker.js` — кеширование App Shell
- `icons/icon-192.png`, `icons/icon-512.png` — иконки

## Запуск локально
1. Откройте `index.html` в браузере. Для полноценного PWA установите на хостинг (см. ниже).

## Деплой на GitHub Pages
1. Создайте репозиторий, например `mini-crm-mobile`.
2. Загрузите все файлы из папки (корень репозитория).
3. В настройках репозитория включите **Pages** → Branch: `main`, Folder: `/ (root)`.
4. Откройте выданный URL. На Android/MIUI появится «Добавить на главный экран».

## Деплой на Netlify
1. Создайте новый сайт из папки проекта (`drag & drop`).
2. Билда нет — статика. Готово.

## Установка на Android (MIUI)
1. Откройте сайт.
2. Меню браузера → **Добавить на главный экран**.
3. Приложение появится с иконкой, запускается полноэкранно.

## Экспорт/импорт
- Экспорт CSV: долгий тап по кнопке «＋» или ПКМ → выгрузит `clients.csv`.
- Импорт не реализован (можно добавить позже).

## Оффлайн
- SW кеширует `index.html`, иконки и манифест.
- Вне сети приложение открывается, данные в IndexedDB/LocalStorage доступны.
